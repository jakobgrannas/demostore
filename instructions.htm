Homework for position as web developer at Vaimo
The Vaimo Frontend Homework

This homework is created for you where you, by using your best practice methods, are going to implement the provided design into a static web page using HTML, CSS and javascript.

The test is used to measure your knowledge and skill level in these techniques and your test will be reviewed by a frontend developer that, among other areas, will look at the following:

  • How well your implementation match the provided design
  • How well written your code is
  • How well structured your code is
  • How optimized your code is
  • How cross-browser safe your implementation is in the current and major prior release of:
    • Internet Explorer (Windows)
    • Chrome (Windows & Mac OS)
    • Firefox (Windows & Mac OS)
    • Safari (Windows & Mac OS)

The PSD as well as the screen-shots for each layer composition is located in the instruction folder within your attached zip file. Please note that all layer compositions also can be found within the PSD.

As you can see, the design consists of a homepage with five additional compositions that show the different interactions a user should be able to do with this page. The design should be centered in the web browser and header and footer backgrounds should be repeatead horizontally, covering the entire width of the browser window.

There are three layer compositions that illustrates a newsletter signup interaction. If the user writes an incorrect email address and presses submit or if the email address is valid a corresponding message, as described in the compositions above, should be shown.

In the compressed folder you received you can see that it also includes a index.html, css-, image and js-folders. If you want, you can use this structure when implementing the design.

Good luck and if you have any further questions just send an email to felead@vaimo.com
